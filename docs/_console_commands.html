Console commands

  • cuicreatepalette name frontcolor [backcolor] - creates a palette in "./Assets/Palettes" from colors
  • printkeys - Prints all MonoGame keyboard keys
  • printcolors - Prints all MonoGame Colors
  • cuipalette - Sets loaded palette as Primary
  • cuipalettedemo - Palette Preview
  • cuicreatepaletteset name primaty secondary tertiary quaternary - Groups palettes in a set
  • cuiloadpaletteset - Load a palette set
  • cuicreateluatypesfile - Manually creates CUITypes.lua in Lua folder

Debug commands:

  • cuidebug - Layout events debug
  • cuimg - Toggles CUIMagnifyingGlass
  • cuidraworder - Prints draw order of all components
  • cuiprinttree - Prints the component tree
  • printsprites - Prints styles in GUIStyle.ComponentStyles
  • cuidebugprops - Opens prop debug
